HTML boilerplate

Boilerplate code or just boilerplate are sections of code that are repeated in multiple places with little to no variation. In VS code when you hit !(Shift+1) with Enter key you can create Html bilerplate.

  • !DOCTYPE html: version 5 of html is written
  • html lang = "en" /html: html tag is top level of element which has two tags(head, body) and it uses a language as English.
  • head: contain information that is processed by machines.
  • meta charset="UTF-8": meta tags describe data of document to the machine and UTF-8 is the standard character encoding you should use in your web pages.
  • meta name="viewport" content="width=device-width, initial-scale=1.0": This tag renders the width of the page to the width of the device's screen size.
  • meta http-equiv="X-UA-Compatible" content="ie=edge": This tag specifies the document mode for Internet Explorer. IE=edge is the highest supported mode.
  • title HTML 5 Boilerplate /title:This text(HTML 5 Boilerplate) is shown in the browser's title bar.
  • link rel="stylesheet" href="style.css": This code will link your custom CSS to the HTML page. rel="stylesheet" defines the relationship between the HTML file and the external stylesheet.
  • body: This is where all the content that will be displayed to users such as the text, images, lists, links, and so on.
  • script src="index.js"/script: External script tags will be placed just before the ending body tag.This is where you can link your external JavaScript code.
